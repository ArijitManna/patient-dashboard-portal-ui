# Patient Dashboard Portal

A modern, responsive React application designed for patients to manage their healthcare information, appointments, and medications.

## Features

### 🏠 Dashboard
- **Vital Signs Overview**: Display current blood pressure, heart rate, temperature, and weight
- **Welcome Section**: Personalized greeting with patient information
- **Quick Stats**: Easy-to-read health metrics in card format

### 👤 Patient Profile
- **Personal Information**: Name, Patient ID, age, and visit history
- **Medical History**: Track of last visit and upcoming appointments
- **Contact Information**: Easy access to patient details

### 📅 Appointments
- **Upcoming Appointments**: List of scheduled visits with doctors
- **Appointment Details**: Date, time, doctor, and appointment type
- **Easy Navigation**: Quick access to appointment information

### 💊 Medications
- **Current Medications**: List of prescribed medications
- **Dosage Information**: Detailed dosage and frequency information
- **Medication Schedule**: Timing and administration details

## Technology Stack

- **React 18**: Modern React with hooks and functional components
- **CSS3**: Custom styling with modern design principles
- **Responsive Design**: Mobile-first approach with breakpoints
- **Modern UI/UX**: Clean, professional healthcare interface

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository** (if applicable):
   ```bash
   git clone <repository-url>
   cd patient-dashboard-portal
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Open your browser** and navigate to `http://localhost:3000`

### Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App (not recommended)
